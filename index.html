Canvas - Fun With Names

By Christopher Dancarlo Danan

I created this project to practice and play around with some canvas stuff I learned online at the MDN website. The canvas is pretty cool and I love that you can do some animations with it.

Each button below does a special effect to the text on the screen. Also, there is a special Easter egg hidden in this project somewhere. If you would like to know how to find it, click here. Click Default button 5 times in a row.

Sorry, but it looks like your browser does not support canvas elements. In order to view this project, you will need to use a browser that is able to use canvas elements.
